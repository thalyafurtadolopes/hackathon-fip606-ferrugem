# 🌿 Hackathon FIP 606 — Ferrugem do Cafeeiro

Este repositório contém os arquivos desenvolvidos para o Hackathon da disciplina **FIP 606**, com análise espacial e modelagem da **ferrugem do cafeeiro** (*Hemileia vastatrix*).

## 📂 Conteúdo

| Arquivo | Descrição |
|---|---|
| `RELATORIOHACKATON.html` | Relatório completo em HTML (abrível em qualquer navegador) |
| `app.R` | Aplicativo Shiny interativo com mapas e análises |
| `Ferrugem14_2.R` | Script R com todas as análises e visualizações |
| `Ferrugem14_2.qmd` | Documento Quarto original |

## 🚀 Como usar

### Relatório HTML
Abra o arquivo `RELATORIOHACKATON.html` diretamente no navegador (Chrome, Firefox, Edge, etc.).

### App Shiny
1. Instale o R e o RStudio
2. Instale os pacotes necessários:
```r
install.packages(c("shiny", "shinydashboard", "leaflet", "sf", "geobr", "ggplot2", "dplyr"))
```
3. Abra o arquivo `app.R` no RStudio e clique em **Run App**
